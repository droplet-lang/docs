# Lists, Map and String

## List
```rust
 // Create a list
let numbers = [1, 2, 3, 4, 5]
println("Original list:", numbers)

// Access elements
let first = numbers[0]
println("First element:", first)

// Modify elements
numbers[2] = 99
println("Modified list:", numbers)

// Empty list
let empty = []
println("Empty list:", empty)
```
